Créer un contrôleur


Sur Play, pour créer un contrôleur, il faut utiliser la classe play.mvc.Controller:
extends Controller
Le contrôleur regroupe l'ensemble des actions.

Action
Une action gère une requête reçue par l'application Play et produit un résultat : play.mvc.Result.
public CompletionStage index() {
    return ok(views.html.board.render(list, form));
}
Plusieurs type de réponse, dont:
  • Result ok = ok("OK :)");
  • Result notFound = notFound();
  • Result badRequest = badRequest(views.html.board.render(list, formWithErrors));
  • Result anyStatus = status(488, "Etrange réponse");
On peut également faire des redirections avec la méthode redirect() en ajoutant la redirection en paramètres.
Pour inserer une redirection vers une méthode d'un contrôleur, on a la notion de routage inverse.


Injection de dépendance

Pour injecter des dépendances on utilise l'annotation @Inject au niveau du contrôleur.
private final Service service;
@Inject
public USController(Service service) {
    this.service = service;
}


Contrôle asynchrone


Sur Play, toutes les requêtes sont gérées de manière asynchrone.
Il faut alors éviter de bloquer le code de l'application.

Pour que le code d'action soit le plus rapide possible, il faut renvoyer la promesse d'un résultat.

Java 8 fournit une API de promesse générique CompletionStage.
Au lieu de renvoyer un Result, il faut envoyer un CompletionStage<Result>.
Cela permet de revenir à notre action sans rien bloquer.

CompletionStage
On a besoin d'une promesse pour créer un CompletionStage<Result>.
CompletionStage promise = computePIAsynchronously();
// Runs in same thread
CompletionStage promiseOfResult = promiseOfPIValue.thenApply(pi ->
    ok("PI value computed: " + pi)
);
Ici, thenApply lance l'execution dans le même thread que la tâche précédente.
Pour pouvoir faire un appel asynchrone, il suffit d'appeler thenApplyAsync.

Si on n'appelle pas de promesse, on peut créer un objet CompletionStage<Result>:
CompletionStage promise = CompletableFuture.supplyAsync(() -> computePIAsynchronously());
